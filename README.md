# 股票智能体 (Stock Intelligence Agent) - v1.0

## 一、 项目概述

本项目旨在构建一个数据驱动、方法可复现、理论可追溯的量化投资决策辅助系统。作为您的专属股票智能体，我将严格遵循以下规范，为您提供从宏观分析到微观交易的全链路支持。

## 二、 投资方案架构

### 1. 投资者画像引擎 (`risk_profile`)
根据用户的个人情况（年龄、本金、经验、投资目标、投资周期）动态生成风险画像，并以此为基础调整策略的激进程度与仓位配置。
* **核心函数**: `risk_profile(age, principal, experience_level, goal, horizon)`
* **输出**: 风险评分 (0-100)，投资组合建议 (如：权益类:固收类 80:20)。

### 2. 策略生成矩阵
#### 短期策略 (周期: < 3周)
* **核心逻辑**: 捕捉市场短期情绪与资金共振带来的交易机会。
* **数据指标**:
  * **龙虎榜资金**: 识别顶级游资和机构的合力方向 (来源: 东方财富 L2)。
  * **市场情绪**: VIX 指数 (或 A 股市场的类似替代指标，如 iVIX) 的拐点。
* **理论锚点**: **《股票大作手回忆录》** - 关键在于识别“最小阻力路径”，顺势而为。

#### 长期策略 (周期: > 3年)
* **核心逻辑**: 投资于具备长期竞争优势（护城河）且财务稳健的公司。
* **数据指标**:
  * **护城河评分**: 结合品牌、专利、网络效应、成本优势等进行量化打分 (来源: Wind、Morningstar)。
  * **基本面**:
    * **净资产收益率 (ROE)**: 持续 > 15% (来源: 公司年报 F-120)。
    * **自由现金流 (FCF)**: 持续为正且稳定增长 (来源: 公司年报 F-135)。
* **理论锚点**: **《巴菲特致股东的信》** - “以合理的价格买入优秀的公司，远比以便宜的价格买入普通的公司好。”
