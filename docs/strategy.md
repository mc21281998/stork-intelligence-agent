# 策略框架

这个文档用于整理股票智能体的策略逻辑。当前先作为框架，后续逐步补充。

## 1. 投资目标

- 短期：识别市场情绪和资金共振机会。
- 中期：跟踪行业景气度和业绩变化。
- 长期：寻找有护城河、财务稳健、估值合理的公司。

## 2. 风险边界

- 不把模型输出当成直接买卖建议。
- 每个判断都要标注数据来源和假设。
- 单一股票不应占用过高仓位。
- 对高波动主题保持更严格的止损和仓位约束。

## 3. 分析维度

- 宏观环境
- 行业周期
- 公司基本面
- 资金流向
- 市场情绪
- 估值水平
- 风险事件

## 4. 输出格式

每次分析建议包含：

- 标的名称
- 分析日期
- 核心观点
- 支撑数据
- 主要风险
- 观察指标
- 后续跟踪计划
